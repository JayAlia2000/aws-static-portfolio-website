# AWS Static Portfolio Website

## Overview
This project demonstrates deployment of a static website using Amazon S3 Static Website Hosting.

The website was built and deployed using AWS cloud infrastructure services and configured through the AWS Management Console.

## Technologies Used
- Amazon S3
- AWS Management Console
- HTML/CSS
- Static Website Hosting
- Bucket Policies
- Cloud Storage

## Features
- Static cloud-hosted portfolio website
- Public website accessibility
- AWS S3 bucket configuration
- Static website hosting enabled
- Custom bucket policy permissions
- Live cloud deployment

## Skills Demonstrated
- Cloud infrastructure fundamentals
- AWS storage services
- Web hosting deployment
- Access management and permissions
- Cloud-based project deployment
- Technical documentation
